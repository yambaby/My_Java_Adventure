LIKE
% - "any number of characters". example: b% = any values that starts with b and have any number of characters after b. doesn't matter if uppercase or lowercase
any position (%b% - any word that has b)
_ - "represents a single character". tells how many number of characters (_b - any two-letter word that ends with b)

SELECT *
FROM customers
WHERE (address LIKE '%TRAIL%' OR 
address LIKE '%AVENUE%') AND
phone LIKE '%4'

REGEXP (regular expression) - searching strings
^ - gets put at the beginning of the string. indicate beginning of a string (^field - any values that starts with field)
$ - gets put at the end of the string. used to find values that ends with the entered string (field$ - words ending with field)
| - multiple search patterns. represents or (field|mac - find values that have field or mac)
[] represents a single character ('[gim]e' - find values that has either ge, ie, or me)
u can also use -(dash) to represent range ('[a-h]e' - any words with an e starting with a-h)

WHERE first_name REGEXP 'ELKA|AMBUR' - first names are ELKA or AMBUR
WHERE last_name REGEXP 'ON$|EY$' - last names end with EY or ON
WHERE last_name REGEXP '^MY|SE' - last names start with MY or contains Se 
WHERE last_name REGEXP 'B[ru]' - last names contain B followed by R or U
